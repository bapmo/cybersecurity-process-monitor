📊 Cybersecurity Process Monitor

A simple and efficient Process Monitoring Tool that provides real-time insights into system processes, helping users track performance and manage resources effectively.

🚀 Features
🔴 Live process monitoring
📈 Real-time CPU & memory usage tracking
📋 List running processes
⚡ Lightweight and fast
🎯 Easy-to-use interface

⚙️ Installation
Clone the repository:
git clone [https://github.com/bapmo/cybersecurity-process-monitor]

Navigate into the folder:
cd cybersecurity-process-monitor

Install dependencies:
npm install
# or
pip install -r requirements.txt

▶️ Usage
Run the application:

npm start
# or
python main.py

🧠 How It Works

The application tracks system processes and updates metrics like CPU and memory usage in real time. It helps users identify heavy processes and manage system performance efficiently.
